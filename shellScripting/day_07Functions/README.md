Functions

Syntax:- 
myFunction() {
 	commands
}

Rules:-
1.No space before ()
2.Curly braces required
3.Function must be defined before calling
4.No semicolon needed if new line


Bash assigns:

Variable	Meaning
$0	--------Script name
$1	--------First argument → Alok
$2	--------Second argument → DevOps
$3	--------Third argument → Linux
$#	--------Number of arguments → 3
$@	--------All arguments (separately)
$*	--------All arguments (as one string when quoted)

Return statement only return from(0-255) not strings

By default all Variables are Global

Local variables are always visible inside the function
